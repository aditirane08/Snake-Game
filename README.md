# Snake-Game
🐍 Snake Game
A modern twist on the all-time favourite Nokia Snake game, reimagined with a rewarding progression system and customizable gameplay.

## 🚀 About the Project

This project is an upgraded version of the classic Snake game, where the traditional gameplay is enhanced with a coin-based reward system and unlockable skins. Instead of just chasing high scores, players now earn coins by eating fruits and use them to **purchase different skins** for both the snake and the fruit, making the game more engaging and interactive.

 ✨ Features
 🎮 Classic Snake Gameplay: Enjoy the nostalgic mechanics of the original Snake game.
 🪙 Coin System:           Earn coins every time the snake eats a fruit.
 🛍️ In-Game Store:        Use collected coins to unlock new skins.
 🎨 Custom Skins:         Multiple skins available for Snake & Fruits
 🔥 Enhanced Engagement:  Adds a layer of strategy and motivation beyond just scoring.
 ⚡ Dynamic Gameplay:     As the game progresses, it becomes more challenging — just like the original.

🧠 Inspiration

This project is inspired by the classic Snake game, where the snake grows longer by eating food while avoiding collisions. 
The goal was to take this nostalgic concept and modernise it with gamification elements like rewards and customization.



## 🎯 How to Play

1. Start the game
2. Control the snake using the arrow keys
3. Eat fruits to:
   * Grow longer
   * Earn coins 
4. Use coins to unlock skins
5. Avoid collisions with:
   * Walls
   * Your own tail
